无障碍声明

Aevia Lab
最后更新：May 1, 2026

1. 我们的承诺

Aevia Lab 致力于让 aevialab.com 为尽可能广泛的受众所用，包括依赖辅助技术的人群。我们力求在合理可行的情况下符合 WCAG 2.1 AA 级标准。

2. 我们的措施

  • 使用语义化 HTML 结构，包含描述性标题、地标和链接文本
  • 保持符合 WCAG 2.1 AA 级正文文本的对比度
  • 为装饰性图片提供替代文本或 aria-hidden 处理
  • 支持全站键盘导航
  • 使用响应式布局，从 320px 视口向上适配
  • 避免自动播放音频和诱发癫痫的动态效果

3. 已知限制

本网站正在积极开发中。某些装饰性 SVG 插图和渐变面板在屏幕阅读器读取时可能无法提供有意义的信息；这些内容已标注 aria-hidden 适当位置。

4. 反馈

如果您在网站上遇到无障碍障碍，或需要其他格式的内容，请通过以下方式联系我们 research@aevialab.com 并描述问题。我们将予以回复，并尽合理努力提供您所需的信息。
